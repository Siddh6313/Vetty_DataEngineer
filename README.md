# Vetty_DataEngineer
This repository contains my SQL solutions for the Vetty Data Engineering Test.
All queries are based on the dataset provided in the assignment and implemented using Oracle 10g.

What This Repository Contains

create_tables.sql – SQL script to create the items and transactions tables
insert_items.sql – Insert statements for the items table
insert_transactions.sql – Insert statements for the transactions table
solutions.sql – SQL queries for all assignment questions

How to Run the Project
Run create_tables.sql to set up the schema
Insert data using insert_items.sql and insert_transactions.sql
Execute queries from solutions.sql
All scripts are fully compatible with Oracle 10g SQL Developer or SQL*Plus.
